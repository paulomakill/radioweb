Rodrigues Radio

Versão 1.0.0 · LilyGO T-Display S3
🌐 Rádio internet via WiFi com interface web
🔊 Controlo de volume por encoder rotativo
📡 Gestão de estações e WiFi pelo browser
🖥️ Display TFT com hora, estação e música
⚠️ Antes de instalar: Use o Chrome ou Edge. Conecte o ESP32 pelo USB. Se a placa não for detetada, segure o botão BOOT enquanto clica em Instalar.
🚀 Instalar Rodrigues Radio ❌ Use o Chrome ou Edge para instalar ❌ Requer ligação HTTPS ou localhost
Após a instalação, ligue ao WiFi RadioConfig (senha: 12345678) e aceda a 192.168.4.1 para configurar.

Depois de configurado, aceda sempre em http://rodrigues.local
